# Affiliate Tweet Generator

Generate 10 tweet options to promote an affiliate product on Twitter/X.

## Input

The user provides a URL (affiliate link). Example:
```
/affiliate-tweets https://imp.i384100.net/k44JM3
```

## Process

1. Fetch the URL to understand the product (name, features, price, target audience, key selling points)
2. Generate 10 tweet options with varied formatting
3. Include the original affiliate link in every tweet
4. Add conversion tips at the end

## Tweet Format Rules

Each tweet must:
- Be under 280 characters (Twitter limit)
- Include the original affiliate URL exactly as provided
- Be copy-paste ready — no placeholders, no "insert link here"

Use these 10 formatting styles, in order:

1. **Single Paragraph** — one clean block of text
2. **3x Bullet Points** — three bullets + short closer
3. **Hook + Contrast** — two short opposing statements
4. **Numbered List** — 4-5 numbered points
5. **Short / Punchy** — 2-3 lines max, minimal words
6. **Question Hook** — opens with a question the reader relates to
7. **Emoji + Bullet Points** — emoji headers + bullets (🧵 opener optional)
8. **Cost Comparison / Social Proof / Myth Bust** — pick what fits the product
9. **Use-Case Angle** — streaming, travel, career pivot, power user — whatever fits
10. **Direct / FOMO / Relatable** — strong closer, pick what fits

## Output Structure

For each option:
```
**Option N — [Style Name]**

[Tweet text with affiliate link]

---
```

End with:
```
**Quick reference:**
- [Which options work best for what]
- [One-line conversion tip]
```

## Tone

- Pragmatic, not salesy
- Australian English
- No AI-isms ("I'd be happy to", "Certainly!")
- Speak like a real person recommending something they've used
