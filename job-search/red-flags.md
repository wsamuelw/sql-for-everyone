# Red Flags in JDs

When reviewing a JD, flag these if present:

- Requires 5+ years in a specific industry (e.g. banking) you don't have
- Must have PhD or specific STEM degree
- On-site 5 days with no hybrid option
- Salary below $150k+ super
- Role is actually 3 jobs in one (e.g. "data engineer + data scientist + BI analyst")
- Vague responsibilities with no clear deliverables
- "Immediate start" combined with a long hiring process (6+ rounds)

Don't block applications for red flags — just flag them so the user can decide.
